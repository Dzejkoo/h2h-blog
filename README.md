# Head2Head - Blog

Live -> [H2H_Blog](https://h2h-blog.vercel.app/)

H2h blog is a single page aplication used: React, GraphQL, NextJS and styled-components.

Blog allows add new post used graphCMS. Also have a lot of features like: comment post, recently post.

# Setup

Clone this repo to your desktop and run npm install to install all the dependencies. Run the app by npm start.
